# Backend

## Getting started

* Create .env file with the following environment variables:
  ```bash
  DATABASE_URL="file:./dev.db"
  # generate with command: openssl rand -base64 60
  AUTH_SECRET_KEY=
  ```

* Run application in dev mode
  ```bash
  yarn dev
  ```
